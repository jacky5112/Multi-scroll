# AI-assisted inverse discovery of 3D/4D multi-scroll attractors and fractional-order robustness assessment

This repository contains the supplementary data, experimental parameters, and high-resolution phase portraits for paper published in Physica Scripta (IOP Publishing).

[![DOI](https://img.shields.io/badge/DOI-10.1088/1402--4896/ae58ff-blue)](https://doi.org/10.1088/1402-4896/ae58ff)

## 📌 Overview
This project utilizes a model-free Deep Reinforcement Learning (DRL) framework, specifically Proximal Policy Optimization (PPO), to inversely discover the system parameters of 3D and 4D multi-scroll chaotic attractors. Furthermore, we assess the topological robustness of these AI-generated attractors under fractional-order derivatives.
